# icm-crew
A centralized library of modular personas and technical components implementing the Interpretable Context Methodology (ICM). This "Crew" leverages folder structure as agent architecture [1] to enable precise context engineering and role-shifting for AI agents.

# This README.md purpose
A guide for others to add new "Crew" members. Include usage instructions etc. This should outline the process for creating new workspaces, perhaps referencing the ICM workspace-builder steps: discovery, stage mapping, scaffolding, questionnaire design, and validation
